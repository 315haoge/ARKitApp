VR(虚拟现实) 是做梦, AR(增强现实) 是见鬼; VR是说谎, AR是吹牛.

### ARKit

ARKit虽然是iOS11新出的框架，但并不是所有的iOS11系统都可以使用，而是必须要是处理器A9及以上才能够使用，苹果从iPhone6s开始使用A9处理器，也就是iPhone6及以前的机型无法使用ARKit

+ A9+
	+ iphone6s 及以后的 iPhone 
	+ iphone SE
	+ ipad pro(A9X)
	+ ipad(2017)

+ IOS11+
